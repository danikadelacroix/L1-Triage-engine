SECTION 5: SOFTWARE REQUIRING APPROVAL (IT-SWR TICKET)
================================================================================

The following categories ALWAYS require an IT-SWR ticket regardless of specific tool:

- Any software not listed in this catalogue
- Open-source tools with GPL, AGPL, or LGPL licensing
- AI and ML tools with data ingestion capabilities (local LLMs, AI coding assistants
  beyond GitHub Copilot, tools that send data to external APIs)
- Browser extensions beyond the pre-approved list (available on intranet.novatech.in)
- Network monitoring or packet capture tools (Wireshark, etc.)
- Remote access tools beyond AnyConnect (TeamViewer, AnyDesk, etc.)
- Any software that requires disabling CrowdStrike or Windows Defender

HOW TO RAISE AN IT-SWR TICKET
1. Go to support.novatech.in > New Ticket > Category: Software > Software Installation Request
2. Provide:
   - Software name and version
   - Official download URL
   - Business justification (why you need it, which project/task)
   - License type (free, paid, open-source — specify license)
   - Data handling: does this software send data externally? (Yes/No)
3. Approval timelines:
   - Standard request: 3 business days
   - Security review required (AI tools, network tools): 7 business days
   - Emergency with business justification and manager sign-off: same day
4. You will receive approval or rejection via email with reasoning.
